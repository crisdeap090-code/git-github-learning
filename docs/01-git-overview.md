# Tổng quan về Git

## Git là gì?

Git là hệ thống quản lý phiên bản phân tán. Git giúp theo dõi những thay đổi của mã nguồn trong quá trình phát triển phần mềm.

## GitHub là gì?

GitHub là nền tảng lưu trữ repository Git trực tuyến. GitHub hỗ trợ quản lý source code, làm việc nhóm, Issue, Pull Request và GitHub Actions.

## Git và GitHub khác nhau

| Git | GitHub |
|---|---|
| Chạy trên máy tính | Hoạt động trực tuyến |
| Theo dõi thay đổi code | Lưu trữ repository |
| Tạo commit và branch | Hỗ trợ Pull Request và Issue |
| Không bắt buộc Internet | Cần Internet để push và pull |

## Một số khái niệm

### Repository

Repository là nơi lưu source code và lịch sử thay đổi của một project.

### Commit

Commit là một phiên bản được lưu lại trong lịch sử Git.

### Branch

Branch là một nhánh phát triển độc lập.

### Remote

Remote là repository được lưu trên nền tảng trực tuyến như GitHub.
